Careers at Acme — Senior Engineer

This is a sample of a customer's own careers page. The apply form below is the Aster widget. Applicants flow straight into the Acme workspace in Aster, parsed and ranked.

Where the values come from

  • data-aster-url and data-aster-key: your Supabase project URL and its public anon key (safe to expose in the browser).
  • data-aster-job: the UUID of the role you posted in Aster. Copy it from the role in your workspace.
  • data-aster-source: any label. It shows on the Applicants page so you can see which channel each applicant came from.
  • data-aster-accent: optional hex colour to match your brand.

Full job board (live list of all open roles)

This renders every open role in the workspace, straight from Aster. New roles appear automatically, and clicking one opens the apply form inline. No hardcoded job UUIDs.

Matching your brand colour

Set data-aster-accent (or accent in the programmatic form) to any hex colour. It drives the buttons, focus rings, and hover states in both widgets. For example data-aster-accent="#0F9D58" for green.

Programmatic version (many roles on one page)

<script src="/embed/apply-widget.js" defer></script>
<div id="role-a"></div>
<script>
  window.addEventListener('DOMContentLoaded', function () {
    AsterApply.mount('#role-a', {
      supabaseUrl: 'https://YOUR-PROJECT.supabase.co',
      anonKey: 'YOUR_SUPABASE_ANON_KEY',
      jobId: 'THE-JOB-UUID',
      source: 'Acme Careers Page',
      onSuccess: function (res) { console.log('applied', res.candidateId, res.fit); }
    });
  });
</script>
